# ML From Scratch (Java)

Implementing machine learning algorithms from scratch in Java.

## Purpose

This repository is a study project to understand machine learning algorithms by implementing them from scratch without external ML libraries.

## Implemented

- [x] Linear Regression
- [x] Gradient Descent
- [x] Bias Optimization
